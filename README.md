# E-Commerce Double Extortion Defense Platform

Advanced functional prototype demonstrating cutting-edge security concepts for e-commerce applications.

## 🎯 Features

- **Real-time Attack Visualization** - Live dashboard monitoring
- **Blockchain Audit Trail** - Tamper-proof logging
- **Advanced Ensemble ML** - Stacking, voting, boosting
- **Agentic AI** - Autonomous decision-making
- **Smart Canaries** - ML-generated honeypot records
- **Differential Privacy** - Data protection

## 🏗️ Architecture

- **Frontend**: React + Material-UI (Deploy to Vercel)
- **Backend**: Flask + SocketIO (Deploy to Render)
- **Database**: MongoDB Atlas (Cloud)
- **ML Models**: Scikit-learn, XGBoost, Isolation Forest

## 🚀 Quick Start

**👉 See [START_HERE.md](START_HERE.md) for complete setup instructions!**

### Prerequisites
- Python 3.9+
- Node.js 18+
- MongoDB Atlas account (free tier)

### Quick Commands

**Backend:**
```bash
cd backend
venv\Scripts\activate  # Windows
python app.py
```

**Frontend:**
```bash
cd frontend
npm run dev
```

**Test Attack (2 minutes):**
See [SIMPLE_ATTACK_TEST.md](SIMPLE_ATTACK_TEST.md)
